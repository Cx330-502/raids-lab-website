内部テスト段階では、ユーザーとアカウントの完全なクォータ機能は一時的に利用できません。現在、ユーザーごとの提出には上限がありませんが、以下の条件に従っています。 FIFO ルールでは列に並ぶことが要求されているので、このセクションに関しては気にする必要はありません。

今後、ユーザーが増えれば、クォータに関する詳細な内容を改善していく可能性があります。
