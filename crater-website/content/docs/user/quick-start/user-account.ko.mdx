내부 테스트 단계에서는 사용자와 계정에 대한 완전한 할당량 기능을 일시적으로 제공하지 않습니다. 현재 각 사용자의 제출은 상한선이 없으며, FIFO 규칙에 따라 대기열에 추가되기 때문에 이 섹션을 신경 쓰지 않으셔도 됩니다.

사용자가 증가함에 따라 우리는 할당량에 대한 세부 사항을 완비할 수 있습니다.
