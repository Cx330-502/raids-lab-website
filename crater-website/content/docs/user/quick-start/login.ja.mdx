## 申請 GPU クラスターアクセス権限

<Callout>

もしあなたが以前に申請したことがあるならば。 Slurm GPU クラスターへのアクセス権がある場合は、このステップをスキップできます。

</Callout>

ログイン [ACT 実験室ユーザープラットフォーム](https://user.act.buaa.edu.cn/)，ここでは会議室の予約や申請をすることができます。 GPU クラスターアクセス権など。

まず、右上のアバター枠をクリックしてプラットフォームにログインします。

その後、トップナビゲーションバーのをクリックしてください。「実験室」，そして、選択してください。「私の情報」。

まだそうでない場合は、 GPU クラスターへのアクセス権がある場合、表示されるインターフェイスは次のようになるはずです。

![GPU クラスターアクセス権限](./images/no-access-to-gpu.webp)

クリックしてください。「使用権限の申請」，申込書に記入し、管理者（杜英杰先生）による審査を待ちます。
