During the internal testing phase, we haven't yet enabled the full quota capacity for users and accounts. Currently, there's no limit on the number of submissions per user, and everything is proceeding as usual. FIFO You just need to queue up according to the rules, so you don't need to pay attention to this section.

In the future, as the number of users increases, we may refine the details of the quota.
