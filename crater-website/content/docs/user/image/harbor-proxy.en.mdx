## Why do we need a mirror repository agent?

#### The connection is stable

For some reasons, it's not possible to directly and stably access some mirror repositories on foreign websites in China (such as <). gcr.io、docker.io etc., laboratory Harbor An agent has been configured, so users can use it to Harbor Use the agent to retrieve the mirror image from the foreign network

#### Avoid being banned

Laboratories may frequently use certain images, but if users pull them too often, they might be banned by some repositories (such as nvcr.io），Therefore, caching to the local device can avoid this problem

#### Accelerate the pull request

For the foreign website mirror, you can access it through Harbor First, download it locally for caching. When users use it, they can retrieve the image faster through the intranet of the laboratory
