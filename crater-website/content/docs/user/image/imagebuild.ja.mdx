## ミラー制作ページに入る

サイドバーの「データとイメージ」カテゴリにある「イメージ管理」メニューの「イメージ作成」をクリックすると、イメージ作成ページに移動できます。

![Image](./img/imagebuild-homepage.webp)

ページにアクセスした後、ページの上部にて、ミラーリング機能を使用して作成したミラーの総数と、あなたが作成したミラーが表示されます。 Harbor ミラーリングされたリポジトリのストレージ容量と制限値を確認できるほか、自分のデータも表示されます。 Harbor プロジェクト名は、こちらで入力できます。 Harbor あなたが作成したイメージを対応する項目で見つけてください。

写真に示された例では、ユーザーが「イメージ作成」機能を利用してイメージを作成したわけではなく、ローカルで作成したイメージを直接アップロードしたため、イメージの総数は0ですが、ストレージ容量は0ではありません。この部分のチュートリアルは「ローカルイメージのアップロード」を参照してください。

画面上部の右側にある「初期認証情報を取得」ボタンを使用して、認証情報を取得します。 Harbor ユーザー名とパスワードを入力すると、直接ログインできます。 Harbor ウェアハウスでは、手動でイメージをアップロードしたり、表示したりできます。 Harbor ウェブサイトのミラーサイト。ただし、このボタンをもう一度クリックすると、パスワードがリセットされるので注意しましょう。

### 3. に基づいて Dockerfile 構築する

高度な自律性が求められるユーザーは、利用するかどうかを選択できます。`に基づいて Dockerfile 構築する`の構築方法。この構築方法は、ユーザーのイメージに関する柔軟な要件にうまく対応することができます。

この構築方法を選択した場合、あなたはミラーの説明と完全かつ文法的に正確なコマンドを記入する必要があるだけです。 Dockerfile（国内のネットワーク問題を考えると、対応するダウンロードやコードの取得時に、ソースの変更が必要になる場合があります。これら両方の項目は必須であり、可能な例は下記の図に示されています。

![Image](./img/imagebuild-dockerfile.webp)

ここで使用されているサンプル Dockerfile 以下をご覧ください。

```dockerfile
FROM gpu-harbor.act.buaa.edu.cn/user-liuyizhou/nvidia-pytorch:24.12-v1.2.1

USER root

RUN pip install onnxruntime-gpu && \
    echo -e '#!/bin/bash\nunzip -o /home/$NB_USER/workspaces/test.zip -d /workspace/test' >> /usr/local/bin/before-notebook.d/prepare_workspace.sh && \
    chmod +x /usr/local/bin/before-notebook.d/prepare_workspace.sh
```

フォームの右下にある「製作を開始」をクリックすると、イメージの製作が開始されます。

## ミラーを確認してください。

「制作を開始」をクリックした後、ページ下部のリストに提出したミラー画像が表示され、初期状態は「待機中」となります。

![Image](./img/imagebuild-result.webp)

ミラーの説明をクリックすると、詳細情報が表示されます。その後、「ビルドログ」タブでミラーのビルドログを確認でき、潜在的な問題を調査しやすくなります。

![Image](./img/imagebuild-log.webp)

また、タブを「<」に切り替えることもできます。Dockerfile”または“Envd 設定」をクリックして、システムがフォームを通じて生成したか、またはあなたが記入した内容を表示します。 Dockerfile または Envd 脚本。

![Image](./img/imagebuild-dockerfile-show.webp)

最後に、イメージの構築に成功した場合、そのページでイメージの状態が「成功」に変わり、作成されたイメージのサイズが表示されます。ただし、注意してください。 Docker 階層型ストレージメカニズムを利用する場合、実際のストレージ容量は各イメージのサイズの合計よりもはるかに小さい可能性があります。同時に、「イメージリスト」であなたが作成したばかりのタイプを表示できます。Custom”のミラー版で、「バッチジョブ」や「インタラクティブジョブ」を開始することができます。
