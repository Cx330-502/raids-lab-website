## プラットフォームの概要

Crater これは北航が独自に開発したクラスター管理プラットフォームであり、クラスターユーザーに効率的で便利なクラスターリソース管理ソリューションを提供します。クラスター内のコンピューティングリソース、ストレージリソースなどを統一的にスケジュール設定および管理でき、クラスターの安定した運用とリソースの最適な利用を確保します。

### 直観的なインターフェイスデザイン

Crater シンプルで使いやすいユーザーインターフェイスが提供され、ユーザーはグラフィカルな操作インターフェイスを通じて、さまざまなクラスター管理タスクを簡単に実行できます。例えば、リソースダッシュボードを使用すると、クラスターリソースの使用状況がリアルタイムで表示され、以下のような機能が利用できます。 CPU 使用率、メモリ占有量、ストレージ容量などの指標。

ジョブ管理インターフェイスでは、実行中のジョブ、ジョブキュー、ジョブ履歴が表示され、ユーザーはいつでもタスクの状態を確認したり、操作を行ったりできます。

![jobshow](./assets/jobshow.webp)

### 効率的なリソース調度

プラットフォームはインテリジェントなリソーススケジュールアルゴリズムを採用しており、ジョブの優先順位やリソース要件などの要因に基づいて、ジョブに最適なリソースを自動的に割り当てます。例えば、複数のジョブが同時にリソースを要求する場合には、Crater 迅速に分析し、最も緊急かつ重要なタスクにリソースを割り当てることで、全体的な実行効率を向上させることができます。

### 監視機能

詳細な監視データとログ記録により、ユーザーはクラスターの運用状況を深く理解することができ、トラブルシューティングやパフォーマンス最適化を容易に行うことができます。

![log_diagnosis](./assets/log_diagnosis.webp)
