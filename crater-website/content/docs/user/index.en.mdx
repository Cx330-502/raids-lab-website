## Platform Introduction

Crater It is a platform independently developed by Beihang University, specifically for cluster management, providing cluster users with an efficient and convenient cluster resource management solution. It can unifiedly schedule and manage computing resources, storage resources, and other resources in the cluster, ensuring the stable operation of the cluster and the optimal utilization of resources.

### Intuitive interface design

Crater It provides a concise and user-friendly user interface, allowing users to easily complete various cluster management tasks through a graphical operation interface. For example, the resource dashboard can display the usage of cluster resources in real time, including CPU Indicators such as usage rate, memory occupancy, and storage capacity.

The job management interface can display running jobs, job queues, and job history records, making it convenient for users to check the task status and perform operations at any time.

![jobshow](./assets/jobshow.webp)

### Efficient resource scheduling

The platform adopts an intelligent resource scheduling algorithm that automatically allocates the optimal resources to tasks based on factors such as the priority of the tasks and the resource requirements. For example, when multiple tasks request resources simultaneously,Crater It can quickly analyze and allocate resources to the most urgent and critical tasks, thereby improving overall execution efficiency.

### Monitoring function

Detailed monitoring data and log records provide users with a basis for gaining a deep understanding of the cluster's operation, making it easier for them to troubleshoot faults and optimize performance.

![log_diagnosis](./assets/log_diagnosis.webp)
