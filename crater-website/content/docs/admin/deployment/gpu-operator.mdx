## 概述

**NVIDIA GPU Operator** 自动部署和管理所有支持 Kubernetes 集群中 GPU 所需的组件。

在 Crater 中，它提供以下功能：

- GPU 驱动安装  
- NVIDIA 容器运行时设置  
- `dcgm-exporter` 用于 GPU 监控（由 Prometheus stack 使用）  
- 与 Crater 的任务调度和 GPU 指标显示的平滑集成

> Crater 需要 GPU Operator 来确保 GPU 任务正确调度和监控。
