Before we officially start, let's first cover some basics about RDMA Related basic knowledge:

- **RDMA**：It is a network communication technology that bypasses the operating system kernel. Its core is to directly access remote memory through the network card, avoiding the traditional approach. TCP/IP The overhead of data copying and context switching in the protocol stack.
- **NVIDIA GPU Direct**[^2]：Implement GPU Video memory and network card DMA The direct connection of the engine, when GPU When it's necessary to communicate with remote nodes, the data can be transmitted directly via InfiniBand Or RoCE The network card transfers data without going through the host's memory for intermediate storage.
- **Network virtualization**：Macvlan And SR-IOV They are two common network virtualization solutions.Macvlan Allows the creation of virtual network card interfaces for containers, making them appear as independent devices on the physical network; and SR-IOV By leveraging the hardware virtualization capabilities of the physical network card, a single physical function can be implemented.PF）It is divided into multiple virtual functions.VF），Every VF All of them can be directly assigned to Pod Use it.
- **Technical path**：At present RDMA There are mainly InfiniBand And RoCE Two implementation methods[^6]。InfiniBand Native support RDMA For the agreement, it is necessary to build an independent network with dedicated switches and subnet managers, which is very costly. However, RoCEv2 Then, based on the traditional Ethernet infrastructure, through PFC And ECN It relies on flow control mechanisms to ensure lossless transmission and is widely used by Internet companies.
