# 潮の心

静かな風の多い銀の港町では、空気中には常に塩っぱさと湿った岩の香りが漂い、そこで暮らすのはリオという名前の少年だった。彼の日々は潮の流れのリズムによって測られ、平凡な生活の中で予測でき、安らぎを与えてくれる節拍器だった。銀の港町は穏やかな日常と海の伝説に満ちており、まるで既に黄ばんで柔らかい白黒写真のような印象を与える場所だった。

爽やかな午後、巨大な潮が海藻や浮き木を混乱させながら岸辺に打ち上げた後、リオはいつものように浜辺を散歩していた。彼は一瞬たりとも忘れられることのないものたちを収集する少年だった——奇形の海ガラス、完璧な渦巻きの貝殻、あるいはサギの足跡のようなもの。その日、彼は通常とは異なるものを発見した。

深緑色の海藻が絡まってできた一塊の中に、石が転がっていた。それはなめらかで灰色で、無数の年月を海の波の中で過ごしてきたものであるはずで、平凡な存在であるはずだった。しかし、一つだけ例外があった。その中心から、静かで安定したリズムで柔らかな内側の光が脈動していた。それは暗い空の反射ではなく、自身の光であり、微かなものではあったが、確かなものだった。

全身にわくわくした感覚と、やや健康的な疑いが伝わった。彼はこれまでにそのようなものを目にしたことも聞いたこともなかった。地元の伝説は美人魚の涙や失われた海賊の黄金の話に満ちていたが、発光する石というものは聞いたことがなかった。彼は慎重にそれを拾い上げた。触ると予想以上に温かく、まるで温かくて生きているような暖かさが手のひらにしみ込んでいた。彼はそれをポケットにしまうと、家に通じる曲がった小路に向かって歩き始めた。

<br />

小さな部屋に戻ったリオは、その発見をより詳しく調査した。薄暗い室内では、石の光がより明確に見えるようになった。それは柔らかな乳白色の光であり、彼がすでに気づいていた脈動は実際に存在し、まるでゆっくりと眠っているような心拍のように感じられた。彼は、これは守らなければならない秘密であることを知った。

彼がその魅力的な光を見つめていると、彼の猫ジャスパが冗談を交えながら彼の布団のほつれた糸に突進し、リオは心から笑い出した。その瞬間、石の中の光が輝き、明るさが明らかに増し、手の中の温かさも強まった。この突然の変化は彼を驚かせた。光は美しかったが、その反応は不安をもたらした。

彼の脳内に科学的な考えと、少しの恐怖が同時に浮かんだ。彼は目を閉じ、想像できる最も悲しい出来事を思い描こうと自分に言い聞かせた——彼が大切にしていた小さな木の船が無情な潮によって奪われた瞬間。その記憶が浮かぶと、石の中の光は弱まり、ほとんど認識できないような光点にまで小さくなった。温かさは完全に消えた。

その日の夜、彼は家に帰った。そして、彼の猫ジャスパが冗談を交えながら彼の布団のほつれた糸に突進し、リオは心から笑い出した。その瞬間、石の中の光が輝き、明るさが明らかに増し、手の中の温かさも強まった。この突然の変化は彼を驚かせた。光は美しかったが、その反応は不安をもたらした。

ニュースが届いた。歴史的な嵐が遠くの海で形成され、大きな風雨の渦が海岸線に向かっており、銀の港町はその経路に位置している。町中には不安の空気が漂い、漁師たちは自分の船を固定し、隣人たちはお互いの助けを借りて木板で窓を塞いで、迫る怒りを防ごうとした。

この広がる恐怖は彼の石にとって毒のように感じられた。リオはその光が弱まっており、脈動も細かく微弱であることを感じ取ることができた。彼がどんなに努力しても喜びを想像しようとすることはできなかった。町全体の不安が彼の小さな喜びの火を窒息させる重たいカーテンのように感じられた。ポケットの中の石は今や冷たく静かだった。

嵐は夕方過ぎにやってきて、町に猛スピードで、耳をつんざくような咆哮をもって襲った。風は復讐の幽霊のように尖った音を立てて吹き、巨大で恐ろしい黒い波が町の古い防波堤にぶつかり、乱れた空中に水しぶきを高々と送り上げた。電気が一瞬光った後、完全に止まり、銀の港町は深く恐ろしい闇の中に包まれた。

家族とリビングに集まり、家の音を風の攻撃に聞かせながら、リオは心の底に絶望的な決意が固まっていた。彼はただの少年であり、嵐の力の前では小さくて無力に感じられた。しかし彼にはその石があった。彼はその冷たい表面を握りしめ、その存在を感じていた。

彼は意識的な選択をした。彼は周囲の恐怖に屈服せず、目を閉じて集中した。彼はこの脆弱で美しい故郷のすべてを愛していることを思い出した。彼はパン職人の微笑み、朝の陽光が港を照らす様子、老いぼれた船乗りたちが語る物語を想像した。彼は愛、希望、勇気のすべてを石に注ぎ込んだ。

その後に起きたことは、彼の記憶の中で永遠に刻まれる純粋で説明できない魔法の瞬間となった。石は彼の絶望的な願いに応え、光が戻った。だが、それは穏やかな脈動ではなく、爆発のようなものだった。石は数秒の間に火花から眩しく輝く灯台へと変化した。金色で慈愛に満ちた暖かい流れが彼の手から迸り、家の壁を通り、町の中心部を包む静かな閃光の保護罩となった。最も猛った風もその見えないシールドの周りで分かれたように感じられた。

翌日の朝、太陽が昇り、奇妙な平和な光景を照らしていた。町の周辺は倒れた木々と廃墟で嵐の暴虐を示していたが、保護罩の下にあった町の中心部はほぼ無傷だった。嵐は過ぎ去った。リオの手にその石はもう温かくも明るくもなかった。それはただの普通で冷たい灰色の卵石に過ぎなかった。その内側の光は完全に消え去っていた。彼は自分の年齢を超えた明確な理解で、魔法は石の中ではなく、*石を通して*導き出されたことを悟った。それは彼自身の心が家族を救ったのだ。彼は水辺に向かって歩き、静かに石を海に返し、教わったすべてに感謝した。彼が家に帰るとき、彼はもはや魔法の秘密を持つ少年ではなく、希望の力の深さを理解した男になっていた。
