# 潮の心

静かで風の強い銀港町。空気にはいつも塩気と湿った岩の匂いが漂い、そこに暮らすのは、リオという小さな男の子だった。彼の日々は、潮のリズムによって刻まれていた。それは、平凡な生活の中で、予測可能で心地よい節拍だった。銀港町は、穏やかな日常と海の伝説に満ちた場所で、まるで古びた白黒写真のように、柔らかく、やわらかな色合いを帯びていた。

ある澄んだ午後、激しい満潮のあと、海藻や浮木が乱雑に打ち上げられた砂浜を、リオはいつも通り散歩していた。彼は、一瞬の出来事や忘れ去られたものの大切さを愛する少年だった——形の不思議な海ガラス、完璧な螺旋を描く貝殻、あるいはトウネンのほんの一瞬の足跡。その日も、彼は珍しいものを発見した。

絡み合った濃い緑の海藻の下に、一块の石が横たわっていた。それは滑らかで灰色で、何百年も海の波に揉まれ続けてきたもの。普通の石に見えるはずだったが、一つだけ例外があった。その中心から、静かで確かなリズムで、柔らかな内発光が脈打っていた。空の陰鬱な反射ではない。それは、石自身の光。弱々しいが、確かに存在する。

興奮と、少しだけの疑念が、全身を駆け巡った。彼はこれまで、このような物を一度も見たことも、聞いたこともなかった。地元の伝説には、人魚の涙や、失われた海賊の金の話がたくさんあるが、光る石の話はどこにもなかった。彼は慎重にそれを拾い上げた。驚くほど温かく、やさしい、生きているような暖かさが手のひらに染み込んでくる。それをポケットにしまい、家へと続く曲がりくねった小道を歩き始めた。

<br />

自分の小さな、遠くの波の音だけが聞こえる屋根裏部屋に戻ると、リオは発見をより丁寧に調べ始めた。薄暗い部屋の中、石の光はより明確に見えた。それは、やわらかな乳白色の光で、先ほど感じた脈動は、確かに実在していた。まるで、ゆっくりと眠っている心臓のように。彼は、これが自分だけの秘密にしなければならないと知った。

その魅力的な光を凝視していると、猫のジャスパーが、彼の布団のほつれた糸を突然飛びついてきた。その仕草に、リオは思わず本気で笑った。その瞬間、石の中の光が一気に輝き、手のひらに伝わる温かさも強まった。突然の変化に、彼は驚いた。光は美しかったが、その反応は不安を誘った。

科学的な疑念と、少しだけの恐怖が、頭をよぎった。彼は目を閉じ、心の中で、思い浮かべられる限りの悲しい出来事に集中した——あのとき、愛していた小さな木の船が、冷酷な潮にさらわれた瞬間。記憶が蘇ったとき、石の中の光は弱まり、ほとんど目立たない小さな光点に縮まり、温かさも完全に消えた。

それから数日間、リオは秘密で真剣な実験を続けた。彼は、この石が自分の心の鏡であることを理解した。優しい言葉や笑い声がもたらす喜びは、石をまるで小さな恒星のように輝かせた。大きな犬が吠えると、恐怖の鋭い痛みが石の光を不規則に閃かせた。彼は、この石が自分の感情を糧にしていることを悟った。

それは、誰にも理解できない形で、彼を理解する静かな友人となった。小さなポケットサイズの伴侶。彼は、この奇妙な物体と深い絆を感じ、まるで責任を背負っているようだった。できるだけ、喜びで「餌」をやるようにした。夏のイチゴの味、日差しの優しさ、母の歌う子守唄の旋律——最も美しい記憶を思い出すたびに、石に心を注いだ。

そして、ある日、知らせが届いた。遠海で歴史的な暴風が発生しており、巨大な暴風雨の渦が海岸線を直撃する予報だった。銀港町は、その経路の真ん中に位置していた。町全体に、不安が広がり始めた。漁師たちは船を固定し、近所の人々は互いに協力して窓を板でふさぎ、迫りくる怒りの嵐に備えた。

その広がる恐怖は、リオの石にとっては毒だった。彼は、その光が弱まり、脈拍が細かく、かすかになったのを感じた。どんなに努力しても、幸せなことを想像することができなかった。町全体の集団的な不安が、重い毛布のように、彼の小さな喜びの炎を窒息させた。ポケットの中の石は、冷たく、静かだった。

夕方、暴風が襲来した。凶悪で轟音を立て、風は復讐の幽霊のようにうなり、巨大で恐ろしい黒い波が、古びた防波堤に激突し、水しぶきを空高く舞い上げた。電気が一瞬点滅したあと、完全に消え、銀港町は深く、恐ろしい闇に包まれた。

家族と一緒に居間で、家が暴風の攻撃に耐えながら軋む音を聞きながら、リオの心の中に、絶望的な決意が固まった。彼はただの少年。暴風の力の前では、小さく、無力に思えた。しかし、彼には、この石があった。彼はそれを、手のひらで強く握りしめた。その表面は、すでに完全に冷たくなっていた。

彼は、意識的に選択した。周囲の恐怖に屈せず、目を閉じ、心を集中させた。彼が愛する、この儚くも美しい故郷のすべてを思い浮かべた。ベーカリーの店主の笑顔、朝日が港を照らす様子、老船乗りたちが語る物語。彼は、すべての愛、すべての希望、すべての勇気を、石に注ぎ込んだ。

そして、次の瞬間、記憶に残る純粋で説明不能な魔法の瞬間が訪れた。石は、彼の絶望的な願いに応えた。光が戻ったが、それは穏やかな脈動ではなく、爆発だった。数秒のうちに、火花から、眩しい灯台のような光にまで変化した。金色で慈愛に満ちた暖流が、彼の手から噴出し、家を通り抜けて外へ広がり、町の中心部を覆う、静かで瞬きする光の膜を作り出した。最も激しい風さえ、その目に見えない守りの壁の周囲で分かれていたように思えた。

翌朝、太陽が昇り、異様な平和な光景が広がっていた。町の郊外は倒れた木々や瓦礫で荒らされていたが、その光の膜に守られていた町の中心部は、ほとんど無傷だった。暴風は過ぎ去った。リオの手の中の石は、もはや温かさも光も失っていた。ただ、普通の、冷たい灰色の小石。その内なる光は、完全に消え去っていた。彼は、年齢を超えた、はっきりとした悟りを得た。魔法は、石の中にあったのではない。それは、*石を通して*導かれたものだった。自分の心が、家を救ったのだ。彼は水辺へ歩き、静かに石を海に返した。すべてを教えてくれたことへの感謝を、黙って捧げた。家へ戻る道を歩くとき、彼はもう、魔法の秘密を持つ少年ではなかった。希望の深遠な力に気づいた、一人の少年だった。
