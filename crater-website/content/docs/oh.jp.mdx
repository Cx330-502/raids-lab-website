# 潮の心

静かな風が吹く銀の港町では、空気中には常に塩の香りと湿った岩の匂いが漂い、リオという名前の少年が住んでいた。彼の日々は潮の流れのリズムによって測られ、平凡な生活の中でも予測でき、安らぎを与えてくれる節拍器だった。銀の港町は穏やかな日常と海の伝説に満ちており、すでに黄ばんで柔らかな白黒写真のように感じられた。

ある爽やかな午後、特大の潮の後、海藻や浮木が岸辺にごちゃごちゃと打ち上げられた後、リオはいつも通りの散歩に出かけた。彼は瞬間的なものや忘れられがちなものを収集するのが好きだった——奇抜な形の海ガラス、完璧な渦巻きの貝殻、またはサギの足跡のようなもの。その日、彼は何か不思議なものを見つけた。

深緑色の海藻が絡まった中で、石が転がっていた。それは滑らかで灰色で、波の下で無数の年月を経て出来上がったもので、何の特徴もなかったはずだった。ただし、一つだけ例外があった。その中心から、静かで安定したリズムで穏やかな内側の光が脈動していた。それは暗い空の反射ではなく、自身から発する光であり、微弱ではあったが、間違いなく存在していた。

全身に興奮と、多少の健康的な疑いが混じった震えが伝わった。彼はこれまでにそのようなものを見たことも聞いたこともなかった。地元の伝説には人魚の涙や失われた海賊の金の話が満ちているが、光る石という話は聞いたことがなかった。彼は慎重にそれを拾い上げた。触ると意外にも温かく、温かくて生きているような暖かさが彼の掌に染み込んでいた。彼はそれをポケットにしまい込み、家へと続く曲がった小路に向かって歩き始めた。

<br />

小さな、遠くの波の音だけが伴う自分の部屋に戻ったリオは、その発見をもっと詳しく調べることにした。暗い室内では、石の光がよりはっきりと見えた。それは優しく乳白色の光で、彼が以前に感じた脈動は確かに存在し、ゆっくりとした眠っているような心拍のように感じられた。彼は、これは自分だけに守らなければならない秘密だと気づいた。

その魅力的な光を凝視していると、彼の猫ジャスパが彼の毛布の端のほつれた糸を遊び心で突こうとして、リオは思わず真剣な笑い声を上げた。その瞬間、石の中の光が輝き、明らかに明るくなり、手の中にあった温かさも強まった。この突然の変化は彼を驚かせた。光は美しかったが、その反応は不気味だった。

科学的な考えと、少しだけの恐怖が彼の頭の中に閃いた。彼は目を閉じ、想像できる最も悲しいことを考えようとした——彼が愛していた小さな木製の船が無情な潮によって奪われた瞬間。その記憶がよみがえったとき、石の中の光は弱まり、ほぼ見えない点にまで小さくなり、温かさも完全に消えた。

次の数日は秘密で真剣な実験の日々だった。リオはこの石が自分の心の鏡であることを理解した。親切な言葉や共有したユーモアによって得られる喜びは、この石を小さな恒星のように輝かせた。大きな犬が彼に向かって吠えると、恐怖の刺し傷が石の光を不規則に揺らす。彼は、この石が自分の感情を食料としていることに気づいた。

この石は彼の静かな知己となり、ポケットサイズの仲間となり、誰も理解できない方法で彼を理解していた。彼はこの奇妙な物に深い絆を感じ、一種の責任さえも感じていた。できるだけのときは、喜びでそれを「育て」、最も美しい思い出を思い浮かべた——夏のイチゴの味、陽射しが顔を撫でる感覚、母の子守唄の旋律。

そして、消息が伝わった。歴史的な嵐が遠海で形成され、巨大な雨と風の渦が海岸線を狙っており、銀の港町はその経路にあった。不安の空気が町全体を覆い、漁師たちは船を固定し、隣人は木板で窓を塞いで、来るべき怒りに備えた。

この無限の不安は彼の石にとって毒のように感じられた。リオはその光が弱っているのを感じ、脈拍が細く弱くなっていることを感じた。彼がどれほど喜びのことを想像しても、石は反応しなかった。町全体の不安は重たいカーテンのように彼の小さな喜びの炎を窒息させた。ポケットの中の石は冷たく静かだった。

嵐は夕方の後に訪れ、この町を猛スピードで、耳をつんざくような咆哮で襲った。風は復讐の幽霊のように尖った音を立て、巨大で悪意に満ちた黒い波が町の古い防波堤に衝突し、空に高々と水しぶきを上げた。電気が一瞬閃き、そして完全に切れて、銀の港町は深い、恐ろしい闇に包まれた。

家族とリビングで、家が嵐の攻撃によって音を立てながら震えているのを聞いていたリオは、心の底で絶望的な決意が固まった。彼はただの少年で、嵐の力の前では小さくて無力に感じられた。しかし、彼にはその石があった。彼はそれをしっかりと握りしめ、今やその表面は完全に冷たくなっていた。

彼は意識的に選択をした。彼は周囲の恐怖に屈服しなかった。彼は目を閉じ、集中した。彼はこの脆弱で美しい家を愛しているすべてのことを思い描いた。彼はパン職人の微笑み、朝の光が港を照らす様子、老海士たちが語る物語を想像した。彼は愛、希望、勇気をすべてその石に注ぎ込んだ。

次に起きたことは、彼の記憶の中で永遠に刻まれる、純粋で説明のつかない魔法の瞬間となった。石は彼の絶望的な願いに応え、光が戻った。だが、それは穏やかな脈動ではなく、爆発的なものだった。数秒のうちに、火花から輝かしい灯台へと変化し、金色の温かさが彼の手から迸り、家の壁を通り越して、町の中心部を覆う静かな光のカーテンとして広がった。最も猛烈な風もその無形のシールドの周りで分離された。

翌日の朝、太陽が昇り、奇妙な平和が広がっていた。町の周辺では倒れた木々や瓦礫が嵐の暴虐を示していたが、保護された町の中心部はほぼ無傷だった。嵐は過ぎ去った。リオの手の中に、石はもはや温かく明るくはなかった。それはただの普通で冷たい灰色の石で、内なる光は完全に消え去っていた。彼は、魔法は石の中ではなく、*石を通して*導き出されたものだと、年齢以上の明晰さで理解した。彼の心が自宅を救ったのだ。彼は水辺に向かい、静かに石を海へと戻し、教わったすべてに感謝した。彼が家へ戻るとき、彼はもはや魔法の秘密を持つ少年ではなく、希望の深遠な力を理解した男になっていた。
